About This Project

This is a fifty-state project which explores the issues of redistricting and apportionment for both the U.S. House of Representatives as well as state legislatures. For each state, we have interactive geographic maps to see the current districting plans and hexagon carograms to visualize the partisan breakdown of the chambers and delegations, as well as detailed information about the redistricting and electoral processes.
